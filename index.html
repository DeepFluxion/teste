Vamos ver como é isso na prática¶

Veja como simples criar um simple grfico com Python

import pandas as pd # importando a biblioteca
ts = pd.Series(np.random.randn(1000), index=pd.date_range("1/1/2000", periods=1000)) # criando os dados com pandas
ts = ts.cumsum() # fazendao soma acumulativa
ts.plot(); # exibindo o grfico

Veja como simples criar um simple grfico com Python - Outro exemplo

df = pd.DataFrame(np.random.randn(1000, 4), index=ts.index, columns=list("ABCD")) # Criando os dados
df = df.cumsum() # Fazendo a soma acumulada
df.plot(); # Exibindo o grfico

O mesmo exemplo mas separando as variveis

df.cumsum().plot(subplots=True, figsize=(6, 10));

Um outro exemplo at aqui nada impressionante

df = pd.DataFrame(np.random.rand(50, 4), columns=["a", "b", "c", "d"])
df["species"] = pd.Categorical(
    ["setosa"] * 20 + ["versicolor"] * 20 + ["virginica"] * 10
)
df.plot.scatter(x="a", y="b", figsize=(7,7));

Um outro exemplo at aqui nada impressionante - Vamos incrementar um pouco

ax = df.plot.scatter(x="a", y="b", color="blue", s=50, label="Grupo 1")
df.plot.scatter(x="c", y="d", color="red", s=50, label="Grupo 2", ax=ax, figsize=(7,7));

Ok at agora nada muito novo, mas e isso - a cor passou a representar uma varavel, em apenas uma linha de cdigo!!!

df.plot.scatter(x="a", y="b", c="c", s=100, cmap='RdYlGn', figsize=(8,8));

Se voc ainda no se convenceu, veja isso - esse grfico permite visualizar as relaes entre quatro variveis, em apenas trs linhas de cdigo!!!

from pandas.plotting import scatter_matrix
df = pd.DataFrame(np.random.randn(1000, 4), columns=["a", "b", "c", "d"])
scatter_matrix(df, alpha=0.2, figsize=(6, 6), diagonal="kde");

Por fim um grfico mais elaborado que utiliza cores, tamanho e mais!!!

import seaborn as sns
