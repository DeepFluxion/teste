Curso Ecogen de Introdução a Análise de Dados com Python¶

Neste curso vamos começar a aprender como usar o Python para analise de dados. Por análise de dados entendemos todo o processo que envolve a extração, manipulação, processamento, visualização e técnicas básicas de estatística para sumarizar e entender grandes massas de dados com o objetivo de extrais insigths úteis para o negócio.
Para executar essas análises teremos de aprender a programar em com a linguagem Python, não se trata de um curso de desenvolvimento de aplicações ou engenharia de software, mas você pode pensar como aprender a interagir com dados e trazer insigths úteis para o negócio a partir das suas análises.

Porque Python ?¶

Python é uma linguagem interpretada modular com uma sintaxe mínima intuitiva que está rapidamente se tornando uma das linguagens mais populares para conduzir pesquisas. Existem inúmeras bibliotecas para diferentes aplicações, possibilitando realizar diversas análises.

Teremos um curso Hands On ou seja você estará em contato com o Python desde a primeira aula. A cada aula teremos um pre work, você recebe o link do Github, lê e executa os códigos no Notebook. Vamos ver como é isso na prática¶

Veja como é simples criar um simple gráfico com Python

import pandas as pd # importando a biblioteca
ts = pd.Series(np.random.randn(1000), index=pd.date_range("1/1/2000", periods=1000)) # criando os dados com pandas
ts = ts.cumsum() # fazendao soma acumulativa
ts.plot(); # exibindo o gráfico

Veja como é simples criar um simple gráfico com Python - Outro exemplo

df = pd.DataFrame(np.random.randn(1000, 4), index=ts.index, columns=list("ABCD")) # Criando os dados
df = df.cumsum() # Fazendo a soma acumulada
df.plot(); # Exibindo o gráfico

O mesmo exemplo mas separando as variáveis

df.cumsum().plot(subplots=True, figsize=(6, 10));

Um outro exemplo até aqui nada impressionante

df = pd.DataFrame(np.random.rand(50, 4), columns=["a", "b", "c", "d"])
df["species"] = pd.Categorical(
    ["setosa"] * 20 + ["versicolor"] * 20 + ["virginica"] * 10
)
df.plot.scatter(x="a", y="b", figsize=(7,7));

Um outro exemplo até aqui nada impressionante - Vamos incrementar um pouco

ax = df.plot.scatter(x="a", y="b", color="blue", s=50, label="Grupo 1")
df.plot.scatter(x="c", y="d", color="red", s=50, label="Grupo 2", ax=ax, figsize=(7,7));

Ok até agora nada muito novo, mas e isso - a cor passou a representar uma varíavel, em apenas uma linha de código!!!

df.plot.scatter(x="a", y="b", c="c", s=100, cmap='RdYlGn', figsize=(8,8));

Se você ainda não se convenceu, veja isso - esse gráfico permite visualizar as relações entre quatro varáiveis, em apenas três linhas de código!!!

from pandas.plotting import scatter_matrix
df = pd.DataFrame(np.random.randn(1000, 4), columns=["a", "b", "c", "d"])
scatter_matrix(df, alpha=0.2, figsize=(6, 6), diagonal="kde");

Por fim um gráfico mais elaborado que utiliza cores, tamanho e mais!!!

import seaborn as sns
sns.set_theme(style="white")
# Load the example mpg dataset
mpg = sns.load_dataset("mpg")
# Plot miles per gallon against horsepower with other semantics
sns.relplot(x="horsepower", y="mpg", hue="origin", size="weight",
            sizes=(40, 400), alpha=.5, palette="muted",
            height=7, data=mpg)
